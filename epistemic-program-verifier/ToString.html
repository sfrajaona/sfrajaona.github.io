toString :: Formula a -> String Source #

The main function takes any formula (FOLFormula or ModalFormula) and returns a string. The output string is formatted to be parsable by Python. The plan is to use SymPy to remove redundant tautologies. The elimination of the annouce logical operator exploded the size of the formula.
