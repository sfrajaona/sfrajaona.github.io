toSBV :: [Var] -> FOLFormula -> FOLFormula -> Predicate Source #

toSBV takes a constraint \(\phi\) (in FOLFormula) and target formula in FOLFormula. The target formula is usually the translation of a ModalFormula \(\tau(\ldots)\) via the Tau module. Both formulas are converted into SBV formula. Variables are converted into SBV symbolic variables (freeInts and freeBools)

toSymbBExpr: convert a boolean expression in BExpr into SBV SBool. variables in the BExpr are replaced by SBV variables created in freeBools
